技术栈：vue2.x + vuex + vue-Router + vux

## Build Setup

``` bash
# 基本同官方的脚手架操作

# install dependencies (不推荐使用cnpm)
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
